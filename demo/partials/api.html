DTOptionsBuilder

These are the methods that can be called to create a datatables options:
  • newOption(): This will create a DTOption.
  • fromSource(sAjaxSource): This will create a DTOption with the given ajax source.

DTOptions

These are the methods that can be called to customized the datatables options:
  • addSource(sAjaxSource): This will add the ajax source to the DTOption.
  • addFnServerData(fn): This will customize the function to fetch the data. By default, it uses the angular $http in order to call the Ajax request.
  • addDataProp(sDataProp): This will add a data property.
  • addOption(key, value): This will add a specific option. See the datatables documentation for the list of options.

DTColumnBuilder

These are the methods that can be called to create a datatable column to display:
  • newColumn(mData): This will create a DTColumn with the attribute of the object to display in the column.

DTColumn

These are the methods that can be called to customized the columns:
  • withLabel(label): Set the label to display in the <thead>.
  • withClass(sClass): Set the CSS class for the column.
  • setVisible(bVisible): Set the visibility of the column.
Note: Of course, these helpers are not mandatory. You can directly pass the datatable options on the element attributes dt-options and dt-columns.
