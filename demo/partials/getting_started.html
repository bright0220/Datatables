Dependencies

The required dependencies are:

  • AngularJS (tested with version 1.2.6)
  • JQuery (tested with version 1.11.0)
  • Datatables (tested with version 1.9.4)

Download

Manually

The files can be downloaded from:

  • Minified JS for production usage
  • Un-minified JS for development

With Bower


$ bower install angular-datatables

Installation

Include the JS file in your index.html file: 

<script src="angular-datatables.min.js"></script>

Declare dependencies on your module app like this: 

angular.module('myModule', ['angularDatatables']);
