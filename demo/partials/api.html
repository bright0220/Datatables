API

$DTDefaultOptions

You can provide default options to set for all your datatables, such as the language, the number of items to display...

Helper/Wrapper API Description
$DTDefaultOptions setLanguageSource(sLanguageSource) Set the default language source for all datatables.
angular.module('myModule', ['datatables']).run(function($DTDefaultOptions) { $DTDefaultOptions.setLanguageSource('/path/to/language'); });
$DTDefaultOptions setLanguage(oLanguage) Set the default language for all datatables.
$DTDefaultOptions setDisplayLength(iDisplayLength) Set the default numbers of items to display for all datatables.
angular.module('myModule', ['datatables']).run(function($DTDefaultOptions) { // Display 25 items per page by default $DTDefaultOptions.setDisplayLength(25); });

DTOptionsBuilder

This service will help you build your datatables options.

Note: Of course, this helper is not mandatory. This helper only constructs a JSON object. You can directly pass the datatable options on the element attributes dt-options.

Helper/Wrapper API Description
DTOptionsBuilder newOptions() Create a wrapped datatables options.
DTOptionsBuilder fromSource(sAjaxSource) Create a wrapped datatables options with initialized ajax source.
DTOptions withOption(key, value) Add an option to the wrapped datatables options. For example, the following code add the option fnRowCallback:
angular.module('myModule', ['datatables']).controller('myCtrl', function($scope, DTOptionsBuilder) { $scope.dtOptions = DTOptionsBuilder.newOptions() .withOption('fnRowCallback', function(nRow, aData, iDisplayIndex, iDisplayIndexFull) { console.log(aData); return nRow; }); });
DTOptions withSource(sAjaxSource) Set the ajax source.
DTOptions withDataProp(sAjaxDataProp) Set the Ajax properties
DTOptions withFnServerData(fn) Set the data server retrieval function.
DTOptions withPaginationType(sPaginationType) Set the pagination type of the datatables
DTOptions withLanguage(oLanguage) Set the language of the datatables. If not defined, it uses the default language set by datables, ie english.
DTOptions withDisplayLength(iDisplayLength) Set the number of items per page to display.
DTOptions withBootstrap() Add bootstrap compatibility.
DTOptions withColReorder() Add ColReorder compatibility.
DTOptions withColReorderOption(key, value) Add option to the attribute oColReorder.
DTOptions withColReorderOrder(aiOrder) Set the default column order.
DTOptions withColReorderCallback(fnReorderCallback) Set the reorder callback function.
DTOptions withColVis() Add ColVis compatibility.
DTOptions withColVisOption(key, value) Add option to the attribute oColVis.
DTOptions withColVisStateChange(fnStateChange) Set the state change function.
DTOptions withTableTools(sSwfPath) Add TableTools compatibility.
DTOptions withTableToolsOption(key, value) Add option to the attribute oTableTools.
DTOptions withTableToolsButtons(aButtons) Set the table tools buttons to display.

DTColumnBuilder

This service will help you build your datatables column options.

Note: Of course, this helper is not mandatory. This helper only constructs a JSON object. You can directly pass the datatable column options on the element attributes and dt-columns.

Helper/Wrapper API Description
DTColumnBuilder newColumn(mData, labe) Create a new wrapped DTColumn.
DTColumn withOption(key, value) Add the option of the column. For example, the following code add the option sContentPadding:
angular.module('myModule', ['datatables']).controller('myCtrl', function($scope, DTColumnBuilder) { $scope.dtColumns = [ DTColumnBuilder.newColumn('foo').withOption('sContentPadding', 'mmm') ]; });
DTColumn withTitle(title) Set the title of the colum.
DTColumn withClass(sClass) Set the CSS class of the column
DTColumn notVisible() Hide the column.
DTColumn notSortable() Set the column as not sortable
DTColumn renderWith(mrender) Render each cell with the given parameter
angular.module('myModule', ['datatables']).controller('myCtrl', function($scope, DTColumnBuilder) { // Data fetched: {gender: 'Mr', firstName: 'foo', lastName: 'bar'} $scope.dtColumns = [ DTColumnBuilder.newColumn('firstName').withLabel('First name').renderWith(function(data, type, full) { return full.gender + ' ' + full.firstName; }); ]; });
